# miniaudio_ffi_example

Demonstrates the usage of `miniaudio_ffi` plugin.

## Features
- Generates a 440Hz Sine Wave in Dart.
- Feeds audio to the native layer using the Pull-Mode Ring Buffer.
- Visualizes real-time buffer latency.

## Getting Started

1. `flutter pub get`
2. `flutter run`

See the main [README.md](../README.md) for detailed documentation and architectural principles.
